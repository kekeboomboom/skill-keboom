# skill-keboom

Public-safe Codex skills.

## Included Skills

- `frontend_brand_system`
- `project-harness-init`

## Install

```bash
cp -R skills/frontend_brand_system ~/.codex/skills/
cp -R skills/project_harness_init ~/.codex/skills/project-harness-init
```

## Use

Invoke a skill in Codex with:

```text
$frontend_brand_system
$project-harness-init
```
